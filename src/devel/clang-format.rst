Clang Format Configuration
==========================

The following :download:`clang-format configuration <clang-format>` reflects the L4Re coding style as much
as possible.

Git Integration
---------------

clang-format provides a `git integration
<https://clang.llvm.org/docs/ClangFormat.html#git-integration>`_ named
git-clang-format that “can be used to format just the lines touched in git
commits”.
